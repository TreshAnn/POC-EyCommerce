# ey-react-boiler-plate

Starter code for React / TypeScript projects ; uses VITE + SWC as transformer

## Installation

### Requirements

- [Node.js v16+](https://nodejs.org/)

```bash
yarn install
```

## Starting the development server

```bash
yarn dev
```

## Building the application

```bash
yarn build
```
